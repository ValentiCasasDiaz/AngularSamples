Ejemplos de for

Para recorrer arrays de objetos:

@for (bookmark of userBookmarks; track bookmark.id) {
  • {{ bookmark.url }}
    • } @empty {
  • Sin favoritos actualmente...
    • }

    Para recorrer Strings como {{title}}, mediante el uso de la posición de la letra:

    @for (char of title; let idx = $index; track idx) {
  • Pos: {{idx}} - Letra: {{ char }}
    • }

    Para recorrer Objetos Iterables, mediante el uso de la posición de los elementos

    @for (entry of myMap; let idx = $index; track idx) {
  • {{ entry[0] }}: {{ entry[1] }}
    • }

    Para saber si es el primer y último elemento recorrido

    @for (bookmark of userBookmarks; track bookmark.id; let first = $first, last = $last) {
  • {{ bookmark.url }} {{first ? '- Es el primer elemento': ''}} {{last ? '- Es el último elemento': ''}}
    • }

    Para saber si el elemento es par o impar

    @for (bookmark of userBookmarks; track bookmark.id; let odd = $odd, even = $even) {
  • {{ bookmark.url }} - Elemento {{ odd ? 'Impar' : '' }} {{ even ? 'Par' : '' }}
    • }

    Para saber cuantos elementos hay en total

    @for (bookmark of userBookmarks; track bookmark.id; let idx = $index; let count = $count) {
  • {{ bookmark.url }} - {{idx + 1}}/{{count}}
    • }
